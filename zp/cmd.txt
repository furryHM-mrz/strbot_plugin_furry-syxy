[分析GitHub用户的近期活动]
雪泷gh_analyze <GitHub用户名>  -生成一张精美的“内卷”报告卡-
卡片展示
当在聊天中发送 GitHub 仓库链接时，机器人会自动识别并发送该仓库的卡片图片。支持以下格式的链接：
https://github.com/用户名/仓库名
https://github.com/用户名/仓库名/issues/123
https://github.com/用户名/仓库名/pull/123
订阅命令
雪泷ghsub 用户名/仓库名 - 订阅指定 GitHub 仓库的 Issue 和 PR 更新
雪泷ghunsub 用户名/仓库名 - 取消订阅指定仓库
雪泷ghunsub - 取消所有订阅
雪泷ghlist - 列出当前已订阅的仓库
默认仓库设置
雪泷ghdefault 用户名/仓库名 - 设置默认仓库，之后在当前会话中使用简化命令
雪泷ghdefault - 查看当前默认仓库设置
查询命令
雪泷ghissue 用户名/仓库名#123 - 查询指定 Issue 的详细信息
雪泷ghissue 用户名/仓库名 123 - 查询指定 Issue 的详细信息（使用空格分隔）
雪泷ghpr 用户名/仓库名#123 - 查询指定 PR 的详细信息
雪泷ghpr 用户名/仓库名 123 - 查询指定 PR 的详细信息（使用空格分隔）
雪泷ghreadme 用户名/仓库名#123 - 查询指定 readme 的信息
雪泷ghreadme 用户名/仓库名 123 - 查询指定 readme 的信息（使用空格分隔）
如果已设置默认仓库或已订阅了单个仓库，也可以直接使用：
雪泷ghissue 123 - 查询默认仓库的指定 Issue
雪泷ghpr 123 - 查询默认仓库的指定 PR
